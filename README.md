# Retina AI - Retinitis Pigmentosa Diagnostic System

Advanced AI-powered medical image analysis for retinitis pigmentosa diagnosis

## Overview

Retina AI is a clinical-grade diagnostic tool that uses deep learning to analyze fundus images and provide instant predictions for retinitis pigmentosa (RP) and related retinal conditions.

## Features

- Instant Diagnosis: Real-time image analysis
- High Accuracy: Multiple trained models
- Professional Interface: Clean, intuitive web interface
- Confidence Scoring: Detailed metrics

## How to Use

1. Upload a fundus image
2. AI analyzes instantly
3. View diagnosis with confidence
4. Download results

## Technical Details

- Models: EfficientNet, ResNet-50, Vision Transformer, Swin Transformer
- Framework: PyTorch
- Interface: Streamlit
- Deployment: HuggingFace Spaces

## Important Disclaimer

For educational and research purposes only. Always consult qualified ophthalmologists for proper diagnosis.

## Supported Formats

- PNG, JPG, JPEG, TIFF
- 512x512 resolution or higher
- Up to 50MB per image
